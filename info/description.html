Find out if the given string has a sub-string "key" inside?

Word Key can be in any case. key, KEY or even KeY

Input: String.

Output: Boolean.

Example:

{% if interpreter.slug == "js-node" %} 
findKey('A simple string') == false
findKey('this string has a key') == true
{% else %} 
find_key('A simple string') == False
find_key('this string has a key') == True
{% endif %}
