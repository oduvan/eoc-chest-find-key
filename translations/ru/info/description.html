Определите, если ли в переданной строке подстрока "key".

Искомая строка может быть в любом регистре. key, KEY или даже KeY

Входные данные: Строка(str)

Output: Булев.

Example:

{% if interpreter.slug == "js-node" %} 
findKey('A simple string') == false
findKey('this string has a key') == true
{% else %} 
find_key('A simple string') == False
find_key('this string has a key') == True
{% endif %}
